ImageUploading

OpenCV

사용자가 검색하고 싶은 제품의 라벨 사진을 업로드 합니다. 그리고 업로드한 사진 속 실제로 필요한 부분은 상품 번호이기 때문에 이 영역만 따로 가져와 저장할 수 있는 로직을 구현해 OCR에서 사용하도록 합니다.

광학문자인식 ocr

Google Vision API

Google Vision API는 이미지에서 텍스트를 감지하고 추출하는 기능을 지원합니다. 사용자가 업로드한 품명 이미지로 부터 텍스트를 추출하여 크롤링에 사용하도록 합니다.

Web Crawling

beautifulsoup

상품의 가격을 비교할 때 이용하는 대표적인 가격비교 플랫폼(네이버 쇼핑, 다나와, 에누리)에서 필요한 데이터(최저가격, 스토어명, url)를 파이썬 라이브러리 중 beautifulsoup을 이용해 가져와 가격 내림차순으로 사용자에게 제공해줍니다.

telegram chatbot service

Telegram api

텔레그램 bot API를 이용해 최저가 알림을 해주는 봇을 생성합니다. 사용자가 원하는 상품번호와 최저가격을 알림봇에게 보내게되면 상품번호, 최저가격 그리고 사용자의 아이디가 DB에 저장됩니다. 주기적으로 해당 상품번호의 최저가격을 가격비교 플랫폼에서 가져와 저장되어 있는 가격과 비교해 같거나 낮은 숫자가 측정되면 사용자에게 알림을 보내줍니다.
