기획의도

온라인 쇼핑이 좀 더 편리하고 비교적 저렴하기 때문에 오프라인보다 온라인으로 상품을 구매하는 경우가 많다. 하지만 직접 착용해보고 구매할 수 없다는 불편함이 존재한다. 그래서 오프라인에서 제품을 확인한 후 온라인에서 구매하는 쇼루밍족이 늘어나고 있는 추세다.

많은 쇼루밍족 사람들은 쇼핑을 할 때 오프라인보다 온라인에서는 더 저렴하게 살 수 있지 않을까? 라는 생각을 하고 구매하고 싶은 상품을 매장에서 직접 착용해 보고 제품의 라벨을 찍거나 메모하여 인터넷으로 검색해 최저가를 찾아보고 구매합니다. 이때 상품의 품번을 찾아 일일이 네이버 쇼핑, 다나와 같은 사이트에 검색하여 가격을 비교하는 수고를 덜고 자동으로 가격 비교 사이트 별 최저가를 알 수 있는 서비스가 있다면 좀 더 편리하지 않을까라는 생각을 하게 되었다.

ImageUploading & OpenCV

Python library

사용자가 검색하고 싶은 제품의 라벨 사진을 업로드 한다. 업로드한 사진 속 실제로 피룡한 부분은 상품 번호이기 때문에 이 영역만 따로 가져와 저장할 수 있는 로직을 구현해 사용자가 업로드 하면 품명이 적혀 있는 영역만 선택하여 사용할 OCR에 사용하도록 한다.

광학문자인식 OCR

Google Vision API

Google Vision API 중 OCR은 이미지에서 텍스트를 감지하고 추출하는 기능을 지원한다. 사용자가 업로드한 품명 이미지로 부터 텍스트를 감지하고 추출하여 크롤링에 사용하도록 넘겨준다.

Web Crawling

Python beautifulsoup

사람들이 상품의 가격을 비교할 때 사용하는 대표적인 가격비교 플랫폼(네이버쇼핑, 다나와, 에누리)을 대상으로 OCR로 출력한 결과 즉, 사용자가 원하는 상품번호를 사용해 최저가격, 스토어명, url을 가져온다.

오픈마켓이나 쇼핑몰이 아니라 왜 가격비교 사이트인가? 제휴사이트에서 제공하는 할인으로 더 낮은 가격으로 제품을 구매할 수 있기 때문에 파이썬 라이브러리 중 beautifulsoup을 이용해 필요한 데이터(최저가격, 스토어명, url)의 위치에서 값을 추출하여 각 플랫폼의 최저가격을 내림차순으로 정렬해 사용자에게 보여줍니다.

telegram chatbot service

Telegram api

텔레그램 사용자라면 맞춤형 최저가 알림 서비스를 이용할 수 있다. 실시간 혹은 주기적으로 가격비교 플랫폼의 최저가격을 모니터링하고, 그것을 텔레그램을 통해 알림받는 서비스

텔레그램 bot API를 이용해 최저가 알림을 해주는 봇을 생성한다. 사용자가 원하는 상품번호와 최저가격을 알림봇에게 보내게되면 상품번호, 최저가격 그리고 사용자의 아이디를 DB에 저장해 둔다. 주기적으로 해당 상품번호의 최저가격을 가격비교 플랫폼에서 가져와 DB에 저장되어 있는 사용자가 지정한 가격과 비교해 같거나 낮은 숫자가 측정되면 사용자의 아이디로 사용자에게 알림을 보내준다.
